ワンタップシール ウェブサイト実装計画書

プロジェクト概要

サービス名: ワンタップシール (1tapseal.com) 目的: NFCタグシールを「ワンタップシール」というブランドとして確立し、QRコードに代わる新しいスタンダードを作る ターゲット: BtoB企業（特に飲食店・ホテル・美術館・病院・観光地） 核心価値: 「かざすだけ」の圧倒的な体験、透明で馴染むデザイン性

書籍から得た重要インサイト

1. QRコードの根本的な問題

  • 認知の問題: 人は自分がQRコードを読まないのに、他人には読むことを期待している
  • 操作の手間: 「カメラ起動→枠に合わせる」という一手間が、すべての障壁を生む
  • 強制されないと読まない: QRコードは「読まされる」もの、ワンタップシールは「使いたくなる」もの

2. 実証された効果

  • 美術館岡田さん: 「展示解説が読みやすかった」回答が前年比2倍
  • クリニック山田さん: Web問診票入力率が30%→80%に向上
  • 高齢者でも使える: 「かざすだけ」なら誰でもできる

3. ターゲット市場の明確化

最優先ターゲット: 飲食店（モバイルメニュー）

  • 固定QRコードを置いている店: 席ごとに備え付けのQRコードがある店舗
  • 問題点:
    • QRコードが汚れる、剥がれる、デザインを損なう
    • 客が「また読み取るのか…」と面倒に感じる
    • 暗い店内では読み取りにくい
  • ワンタップシールの価値:
    • かざすだけ→注文までスムーズ
    • 透明だから高級感を保てる
    • 夜光シールと組み合わせで暗い店内でも機能
    • 汚れにくく、耐久性が高い

第二優先: 美術館・博物館

  • 高齢者来館者が多い→「かざすだけ」が効果的
  • 展示解説の多言語対応
  • デザイン性重視の空間→透明シールが最適

第三優先: 病院・クリニック

  • 高齢患者が多い→Web問診票の入力率向上
  • 受付業務の効率化

その他: ホテル・旅館、観光地、駅、イベント会場

4. 除外するターゲット

  • カフェの注文用QRコード: 可変QRコード（都度生成）を使っている店は対象外
  • 大量印刷が必要な用途: チラシ・ポスターの大量配布にはQRコードが適している

製品スペック

基本仕様 - サイズ: 直径3cm（円形） - 素材: PET（透明フィルム） - 印刷: 黒色スマホアイコン - チップ: NTAG213（180バイト URL記録容量） - 対応デバイス: - iPhone XS以降（iOS 13以降） - Android（NFC機能搭載機種） - 読み取り距離: 約1〜3cm - 耐久性: 防水加工（IPX4相当）、耐久温度 -25℃〜70℃ - 粘着: 強粘着タイプ

透明シールの3つの特徴

1. デザインに溶け込む - 貼った場所の素材感・色を活かす - 高級感を損なわない - どんなインテリアにも馴染む

2. 視認性と美観の両立 - 黒いスマホアイコンで「ここにかざす」が一目でわかる - でも、透明だから目立ちすぎない - 控えめだが、機能は十分

3. 組み合わせの自由 - 下に目立つ色のシール（直径3cm以上）を貼れば、透明なので下の色がそのまま見える - 夜光・蛍光シールとの併用で暗い場所でも機能 - デザインの自由度が無限大

対応デバイス

iPhone - iPhone XS以降（2018年以降のモデル） - iOS 13以降 - 自動読み取り対応（アプリ不要） - ロック画面からでも読み取り可能

Android - NFC機能搭載機種（2018年以降のほぼ全機種） - Android 5.0以降推奨 - 一部機種ではNFC設定をONにする必要あり

読み取り方法 - iPhone: 画面上部（カメラ付近）をシールに近づける - Android: 背面中央をシールに近づける

URLの書き込み

書き込み可能なURL - https://から始まるURL - 短縮URL推奨（bit.ly、tinyurlなど） - 長いURLも対応可能（180バイトまで）

動的リダイレクトの活用 - シールに書き込むURLは固定 - 短縮URLサービスで飛び先を後から変更可能 - 例: bit.ly/yourshop → 飛び先は自由に変更

書き換え防止 - 出荷時にロック処理済み - 第三者による書き換え不可 - セキュリティ面でも安心

なぜ今、置き換えるべきか

4つの決定的な違い

1. 体験の差 - QRコード: 「読まされる」「やらされる」 - ワンタップシール: 「使いたくなる」「試したくなる」

2. 成功率の差 - QRコード: 角度・距離・照明に敏感、失敗することがある - ワンタップシール: 近づければほぼ100%成功

3. 印象の差 - QRコード: 「ちょっと古い」「安っぽい」 - ワンタップシール: 「先進的」「スマート」「高級感」

4. 利用率の差 - QRコード: 30〜50%（推定） - ワンタップシール: 80%以上（実証済み）

6. 技術情報ページ

NFCとは？

Near Field Communication（近距離無線通信） - 約1〜3cm以内の近距離で通信する技術 - Suica、PASMO、Apple Pay、Google Payと同じ技術 - 国際規格ISO/IEC 14443に準拠 - 安全性が高い（近距離でしか通信しない）

ワンタップシールの仕組み 1. NFCチップにURLが書き込まれている 2. スマホをかざすとNFCチップが電波を受信して起動 3. 自動的にブラウザが起動しURLを開く 4. アプリのインストール不要

対応デバイス

iPhone - 対応機種: iPhone XS以降（2018年以降のモデル） - 対応OS: iOS 13以降 - 特徴: - 自動読み取り対応（バックグラウンドでも読み取る） - ロック画面からでも読み取り可能 - アプリ不要

Android - 対応機種: NFC機能搭載機種（2018年以降のほぼ全機種） - 対応OS: Android 5.0以降推奨 - 注意: 一部機種ではNFC設定をONにする必要あり

対応確認方法 - iPhone: 設定 → 一般 → 情報 でモデルを確認 - Android: 設定 → 接続 → NFC で機能の有無を確認

読み取り方法 - iPhone: 画面上部（カメラ付近）をシールに近づける - Android: 背面中央をシールに近づける

セキュリティ・プライバシー

書き換え防止 - 出荷時にロック処理済み - 第三者による書き換え不可 - 貼り替えの心配がない場合は安心

通信の安全性 - 近距離（約1〜3cm）でしか通信しない - 盗聴・傍受のリスクが極めて低い - HTTPS通信と組み合わせることで完全な安全性確保

個人情報の扱い - シール自体には個人情報を保存しない - URLのみを保存 - URLの飛び先でのデータ管理はお客様側で実施

まとめ

この計画書に基づいて、ワンタップシール（1tapseal.com）のウェブサイトを構築します。

核心価値 - 「かざすだけ」の圧倒的な体験 - 透明で馴染むデザイン性 - 誰も取り残さない優しさ

最優先ターゲット - 飲食店（席に固定QRコードを置いている店） - 美術館・博物館（高齢者来館者が多い） - 病院・クリニック（高齢患者が多い）

差別化要素 - QRコードより圧倒的に簡単（「かざすだけ」） - 透明だから高級感を保てる - 実証データあり（入力率30%→80%など）

実装アプローチ - フェーズ1: MVP（2〜3週間） - フェーズ2: コンテンツ充実（2〜3週間） - フェーズ3: 高度な機能（3〜4週間） - フェーズ4: 最適化・拡張（継続的）

次は、実装を開始しましょう。
